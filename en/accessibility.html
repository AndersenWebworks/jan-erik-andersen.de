Accessibility Statement

Last updated: November 15, 2025

Commitment to Accessibility

Jan-Erik Andersen is committed to making the website jan-erik-andersen.de accessible in accordance with the German Accessibility Enhancement Act (Barrierefreiheitsstärkungsgesetz - BFSG) and the Web Content Accessibility Guidelines (WCAG 2.1) at Level AA.

Conformance Status

This website is substantially conformant with the WCAG 2.1 Level AA standards. Keyboard smoke tests (November 2025) confirmed that all essential interactions are operable.

  • Perceivable: High contrast ratios (text: 17.40:1, accents: 5.89:1), semantic HTML, clear heading hierarchy
  • Operable: Full keyboard navigation, skip link to main content, focus outlines, and a native checkbox-based dark-mode toggle
  • Understandable: Clear language (lang="en"), logical content structure, consistent navigation
  • Robust: Valid HTML5, structured data (JSON-LD), screen reader compatibility

Technical Specifications

The accessibility of this website is based on the following technologies:

  • HTML5 (validated)
  • CSS3 with dark mode support (prefers-color-scheme)
  • JSON-LD for structured data
  • ARIA attributes for enhanced semantics
  • No JavaScript required

Known Limitations

The current iteration intentionally stays minimal:

  • No images or media assets, so no alt text/caption requirements
  • No forms or complex widgets beyond the CSS-only accordions
  • The manual dark-mode toggle uses a native checkbox and works without JavaScript
