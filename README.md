# jan-erik-andersen.de

Personal-Brand-Website von Jan-Erik Andersen / Andersen Webworks.
Webentwicklung, digitale Tools und KI-Sichtbarkeit. Seit 1999.

## Live

https://jan-erik-andersen.de (GitHub Pages, main = live)

## Struktur

```
/                          Startseite (Executive Summary + Daten)
/leistungen/               Drei Service-Cluster mit Preisrahmen
/leistungen/ki-sichtbarkeit.html   GEO-Vertiefung mit Quellenangaben
/projekte/                 Referenzen (Runden Group, Gerwing, Casarotex, SiS, TNI)
/ueber/                    Bio, Disziplinen, Ablauf
/kontakt/                  Kontaktdaten
/en/                       Englische Version
/de/                       Impressum, Datenschutz, Barrierefreiheit
/ai/                       AI-Endpoints (Schema.org, JSON-LD, llms.txt)
```

## Stack

Statisches HTML + CSS. Kein Framework, kein Build-Tool, kein CMS.
Inter Font, AW-Rot (#d40235), Dark Mode, WCAG 2.1 AA.

## Deployment

```bash
git push origin main
# Live nach 1-2 Minuten (GitHub Pages)
```
