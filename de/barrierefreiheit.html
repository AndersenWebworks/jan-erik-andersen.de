Erklärung zur Barrierefreiheit

Stand: 15. November 2025

Bekenntnis zur Barrierefreiheit

Jan-Erik Andersen ist bestrebt, seine Website jan-erik-andersen.de im Einklang mit dem Barrierefreiheitsstärkungsgesetz (BFSG) und den Web Content Accessibility Guidelines (WCAG 2.1) auf Level AA barrierefrei zu gestalten.

Konformitätsstatus

Diese Website ist weitgehend konform mit den WCAG 2.1 Level AA Standards. Tastaturtests (November 2025) bestätigen die Bedienbarkeit sämtlicher Navigationselemente.

  • Wahrnehmbar: Hohe Kontrastverhältnisse (Text: 17.40:1, Akzente: 5.89:1), semantisches HTML, klare Überschriftenhierarchie
  • Bedienbar: Skip-Link zum Hauptinhalt, vollständige Tastaturnavigation, sichtbare Fokus-Indikatoren sowie ein per Checkbox steuerbares Dark-Mode-Element
  • Verständlich: Klare Sprache (lang="de"), logische Inhaltsstruktur, konsistente Navigation
  • Robust: Valides HTML5, strukturierte Daten (JSON-LD), Kompatibilität mit Screenreadern

Technische Spezifikationen

Die Barrierefreiheit dieser Website basiert auf folgenden Technologien:

  • HTML5 (validiert)
  • CSS3 mit Dark-Mode-Support (prefers-color-scheme)
  • JSON-LD für strukturierte Daten
  • ARIA-Attribute für erweiterte Semantik
  • JavaScript optional (Kerninhalt funktioniert ohne JS, interaktive Features wie der Berater-Funnel benötigen JS)

Bekannte Einschränkungen

Aus den internen Audits ergeben sich aktuell keine blockernden Einschränkungen. Hinweise:

  • Bilder haben Alt-Texte, keine Videos vorhanden
  • Keine Formulare → keine zusätzlichen ARIA-Anreicherungen erforderlich
  • Dark-Mode-Umschaltung erfolgt per nativem Checkbox-Element, JavaScript wird nur für die Persistenz (localStorage) verwendet
